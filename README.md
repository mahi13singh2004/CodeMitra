# CodeMitra

AI-Powered Coding Interview Assistant Chrome Extension

## Features

- 💡 **Hint**: Get hints without revealing solutions
- 🐞 **Debug**: Find bugs in your code
- 📖 **Explain**: Understand code and approaches
- ⚡ **Complexity**: Analyze time/space complexity
- 🧠 **Pattern**: Identify DSA patterns and similar problems

## Tech Stack

**Frontend**: React, Tailwind CSS, Chrome Extension APIs  
**Backend**: Node.js, Express, MongoDB, Redis  
**AI**: Google Gemini API  

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Extension
```bash
cd extension
npm install
npm run build
```
